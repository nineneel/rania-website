# Color Tokens

## Primary Colors

```css
--primary-dark: #1A243D;
--primary-gold: #A38447;
--primary-beige: #E6E2D3;
```

## Text Colors

```css
--text-white: #FFFFFF;
--text-light-gray: rgba(0, 0, 0, 0.20);
```

## Gradients

```css
--gradient-dark: linear-gradient(180deg, rgba(26, 36, 61, 0) 34%, #1A243D 100%);
--gradient-black: linear-gradient(90deg, #1A243D 51%, black 100%);
```
