# Spacing Tokens

## Container Widths

```css
--container-width: 1440px;
--content-width: 1023px;
```

## Section Spacing

```css
--section-padding: 60px 209px;
--section-padding-vertical: 60px;
--section-padding-horizontal: 209px;
```

## Component Spacing

```css
--card-gap: 30px;
```

## Border Radius

```css
--border-radius-card: 16.96px;
--border-radius-button: 100px;
```

## Responsive Breakpoints

```css
/* Tablet (768px - 1024px) */
--section-padding: 40px 60px;

/* Mobile (< 768px) */
--section-padding: 30px 20px;
--card-gap: 20px;
```
